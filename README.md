
  eskeli-search 环境版本说明 : 
---------------------------------------------------------------------
  
  JDK Version >= 1.8（ 推荐 ）
  ElasticSearch Version >= 7.x（ 推荐 ） 
  Maven Version >= 3.5.0（ 推荐 ）
  
  由于 eskeli-search 基于 elasticsearch，故 branch 与 elasticsearch version 保持一致，
  以便两者更好兼容
